Política de Accesibilidad

En FinEs Mapa Interactivo, nos comprometemos a garantizar que nuestro sitio web sea accesible para todas las personas, incluyendo aquellas con discapacidades. Nos esforzamos por cumplir con las Pautas de Accesibilidad al Contenido Web (WCAG) 2.1 de nivel AA, impulsando una experiencia inclusiva para todos nuestros usuarios.

Nuestro Compromiso

Trabajamos continuamente para mejorar la accesibilidad de nuestra plataforma. Nuestras acciones incluyen:

  • Utilizar HTML semántico para estructurar el contenido de manera lógica.
  • Asegurar contrastes de color adecuados para el texto y los elementos interactivos.
  • Proporcionar texto alternativo descriptivo para todas las imágenes informativas.
  • Hacer que toda la funcionalidad sea accesible a través del teclado.
  • Ofrecer subtítulos y transcripciones para contenido multimedia (cuando aplique).
  • Diseñar formularios claros con etiquetas asociadas y mensajes de error útiles.
  • Garantizar una navegación consistente y predecible en todo el sitio.
  • Implementar estados de foco visibles para todos los elementos interactivos.

Áreas de Mejora y Retroalimentación

Somos conscientes de que la accesibilidad es un proceso continuo. Estamos constantemente buscando soluciones para llevar todas las áreas de nuestro sitio a los mismos niveles generales de accesibilidad web.

Si encontrás alguna dificultad para acceder a algún contenido o tenés sugerencias sobre cómo podemos mejorar la accesibilidad de nuestro sitio, por favor no dudes en contactarnos. Tu feedback es muy valioso para nosotros.
